# <img src="logo/666.svg" width="30" height="30" /> 海航666-航班/航线可视化工具 <img src="logo/2666.svg" width="30" height="30" />

基于 HTML 的交互式航线可视化工具，用于展示和查询不同机场之间的航线信息。为减少依赖，航班数据通过 python 脚本全部写入 html 中，造成文件相对较大。本地 HTML 需要网络加载地图，但是交互不依赖后端服务器。
由于我不是科班出身，也没有系统学过前端开发，代码由 claude / gemini 撰写。比较粗糙，仅为满足个人需求而作。不过考虑到可能对其他人有用，特此整理后发布。如果有大佬想进一步升级，如有需要，可以通过 Issue 或邮件联系我，尽力协助。海航不公布表格数据后，数据更新需要与其他开发者合作，可能时间周期较长，但会不断优化界面。

## ✈️ 功能介绍
- **移动端支持以及自动识别系统深浅设置**
  - 支持小屏、竖屏等移动端显示
  - 新增暗黑模式（不建议配合卫星地图使用）

- **航线按实际路径绘制**
  - 航线描绘的是实际飞行路线，所以存在经停的情况。
  - 每条航线通过多个箭头显示方向，使用弧线避免重叠。

- **按日期和星期筛选**
  - 可筛选任意某天/多天的航班
    （如仅查看周五、周日航班，或近查看4.30和5.5航班）。

- **按早晚班筛选**
  - 早班与晚班航班使用不同颜色表示。
  - 支持筛选早班或晚班航班。

- **点击机场查看进出港**
  - 点击任意机场，即可查看该机场所有到达和出发的航班。
  - 支持筛选任一机场到达或出发的航班。

- **点击航线查看详细信息**
  - 可查看该航线的所有航班信息。

- **按航司筛选**
  - 支持按航空公司筛选显示航线。
 
- **按机型筛选**
  - 支持按机型筛选显示航线。
  - 支持按大(A330 Series & B787 Series)、中(B737 Series & A320 Series)、小(ERJ Series & C909)选择机型

## 🗺️ 地图来源
本项目基于 [Leaflet.js](https://leafletjs.com/) 地图框架构建。使用了 [CARTO](https://carto.com/) 提供的开源底图服务。部分老版本使用 [OpenStreetMap] / [高德地图] 作为底图。

## ⚠️ 注意事项

- 本地 html 文件所有交互逻辑基于浏览器端运行，无需服务器支持，但需要网络加载地图底图。
- `CN270_cityairport_name_IATA_ICAO_coords.csv` 是自己整理的 中国大陆270机场简名-全名-IATA-ICAO-坐标对照表，将持续更新，目前数据截至日期（2025.12.31），来源于CAAC公布的2025年全国民用运输机场生产统计公报，包括两个取证未运行机场。
- 一些数据把起降地写成了起飞城市所在地区，比如“遵义”实际是指的“遵义新舟”，“茅台”是指的“遵义茅台”......可能存在未知的映射错误，欢迎指出。
